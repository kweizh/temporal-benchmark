## What happens on `npm start`

1. **Build** — `tsc` compiles all TypeScript to `dist/`
2. **Worker** — spawned as a background child process; connects to Temporal Cloud via TLS + API key; webpack-bundles the workflow; starts polling `hello-world-ts`
3. **Client** (after 4 s grace period) — connects to Temporal Cloud; starts `HelloWorkflow` with `workflowId = hello-wf-${ZEALT_RUN_ID}` and arg `"Temporal"`; waits for completion
4. **Output** — prints `Hello, Temporal!` to stdout
5. **Cleanup** — sends SIGTERM to the worker; exits 0
