This task exercises Temporal's Update message-passing primitive via `defineUpdate` + `setHandler` (with `validator`), combined with a Signal to terminate the workflow cleanly. Temporal Cloud credentials are already provided to the environment as the following variables: `TEMPORAL_API_KEY`, `TEMPORAL_ADDRESS`, and `TEMPORAL_NAMESPACE`.
